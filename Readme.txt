#  Personal Portfolio Website

Welcome to my **Personal Portfolio Website**!  
This project showcases my work, skills, and background as a tech enthusiast. It includes key pages such as About, Resume, Portfolio, Services, and Contact.

---
##  Description of Each File

- index.html: The landing/home page introducing me and my site.
- about.html: A detailed page about who I am, my journey, and my interests.
- resume.html: Displays my qualifications, experience, and technical skills.
- portfolio.html: A showcase of the projects I’ve worked on with brief descriptions.
- services.html: Lists the services I can provide or plan to offer.
- contact.html: Allows visitors to reach out to me through a form or contact information.
- assets/: This folder contains all images, CSS styles, icons, and other assets used across the site.

##  Technologies Used

- HTML5 – For creating the structure of web pages  
- CSS –  for styling and layout
- JavaScript and php
